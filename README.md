# libadt

## setup

```
$ git clone --recursive https://github.com/misebox/libadt.git && cd libadt
```

## make static library and headers

```
$ make
```

## executable sample

```
$ make sample
```

## prepare for testing

```
$ git submodule init
$ git submodule update
```

## testing

```
$ make test
```

## coverage

```
$ make cov
```
